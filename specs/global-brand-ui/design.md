# 银发纪全局品牌 UI 设计

## 视觉方向

- 风格：柔和自然的编辑式界面，强调亲切、清楚和稳定。
- 主色：雾蓝 `#DCEEF5`、品牌蓝 `#79BFD8`、深青 `#24566A`。
- 辅色：暖米白 `#F7F2E8`、提示金 `#D99A5B`、白色 `#FFFFFF`。
- 字体：使用微信小程序原生中文字体能力，正文不低于 32rpx。
- 布局：首页采用左文右标的非对称横幅；内页采用左对齐标题、分组卡片和大触控区。

## 页面方案

### 首页

- 顶部由普通容器调整为原生 `swiper`，当前仅放置一张品牌横幅。
- 快捷入口继续保留“看教程、约课程、我的预约”。
- 主内容标题改为“教程推荐”，复用现有教程列表接口，当前取前 3 篇作为推荐；每张卡使用教程 `_id` 直接进入对应详情。
- 后续研发部可将教程截取规则替换为推荐算法，并将 `swiper-item` 改为新闻数据循环；保持现有字段和卡片结构即可。

### 用户主链路

- 教程、课程、预约、“我的”统一背景、边框、按钮、状态色和标题层级。
- 保留现有 WXML 数据绑定、事件名称与导航目标。

### 管理端

- 仅统一品牌色、按钮可读性与危险操作区分，不扩展管理能力。

## 技术边界

- 仅修改 WXML、WXSS 和交付文档。
- 不修改 JS、JSON、service、API、云函数或数据库。
- 使用原生组件与现有本地 Logo 素材，不引入外部依赖。

## 验证

- 检查 WXML 标签闭合、资源路径、事件绑定和导航目标。
- 检查字号、按钮高度、禁用态、空态和错误态。
- 尝试使用微信开发者工具编译；若工具不可用，记录限制并执行静态回归。
