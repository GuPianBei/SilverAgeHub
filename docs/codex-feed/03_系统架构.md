# 系统架构

项目采用微信小程序 + 云开发架构：

```text
微信小程序前端
  ↓
云函数 Cloud Functions
  ↓
云数据库 Cloud DB
  ↓
云存储 Cloud Storage
```

## 前端模块

- 用户端：首页、教程、课程、预约、我的
- 管理端：教程管理、课程管理、预约管理

## 云函数模块

- `guide`：教程列表、详情、新增
- `course`：课程列表、详情、新增
- `order`：预约创建、预约列表
- `admin`：管理端通用查询、更新、删除

## 数据集合

- `guide`
- `course`
- `order`
