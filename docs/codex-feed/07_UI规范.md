# UI 适老化规范

银发纪面向老年用户，界面优先保证清晰、直接、可点击。

## 基础原则

- 正文字号不低于 16px，小程序中建议使用 `32rpx` 起
- 重要按钮高度不低于 `88rpx`，当前项目统一接近 `96rpx`
- 页面文案简短，避免复杂术语
- 高对比度，背景和文字区分明显
- 操作路径少，首页直接给出教程和课程入口

## 页面设计

- 首页突出「看教程」「约课程」
- 教程详情优先展示步骤内容，视频字段作为增强能力
- 课程详情清晰展示时间、地点、人数
- 预约表单只保留必要字段：姓名、电话
- 我的页面只展示预约记录与状态

## 不做内容

- 不加入社交动态流
- 不加入聊天入口
- 不加入复杂推荐解释
