# 云函数设计

云函数按业务资源拆分，前端通过 service 层调用。

## guide

- `list`：获取教程列表
- `detail`：获取教程详情
- `create`：新增教程

## course

- `list`：获取课程列表
- `detail`：获取课程详情
- `create`：新增课程

## order

- `create`：提交课程预约，默认状态为 `未确认`
- `list`：获取当前用户预约记录

## admin

- `list`：管理端查看列表
- `update`：更新教程、课程或预约状态
- `delete`：删除教程或课程

管理端写操作后续需要接入管理员身份校验，MVP 阶段先保留基础能力。
