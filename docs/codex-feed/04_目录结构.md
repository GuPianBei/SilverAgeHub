# 项目目录结构

```text
.
├── miniprogram/                 # 微信小程序前端
│   ├── pages/                   # 页面
│   │   ├── index/               # 首页
│   │   ├── guides/list/         # 教程列表
│   │   ├── guides/detail/       # 教程详情
│   │   ├── courses/list/        # 课程列表
│   │   ├── courses/detail/      # 课程详情
│   │   ├── orders/create/       # 预约表单
│   │   ├── mine/index/          # 我的预约
│   │   └── admin/index/         # 管理端入口
│   ├── services/                # 前端 service 层
│   ├── utils/                   # 工具与云环境配置
│   ├── components/              # 复用组件
│   └── assets/                  # 图片等静态资源
├── cloudfunctions/              # 云函数
│   ├── guide/
│   ├── course/
│   ├── order/
│   └── admin/
├── database/                    # 数据库 schema 与种子数据
├── admin/                       # 独立管理端说明/后续入口
└── docs/                        # 产品文档与投喂资料
```
