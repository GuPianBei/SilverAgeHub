# 数据库设计

MVP 使用 3 个云数据库集合：`guide`、`course`、`order`。

## guide 教程表

- `title`：教程标题
- `category`：教程分类，例如微信使用、防诈骗、生活服务
- `content`：教程步骤内容，建议使用 JSON 数组保存
- `images`：图片云存储地址数组
- `video`：视频云存储地址
- `createTime`：创建时间

## course 课程表

- `title`：课程名称
- `description`：课程介绍
- `time`：上课时间
- `location`：上课地点
- `limitCount`：人数上限
- `createTime`：创建时间

## order 预约表

- `userName`：预约人姓名
- `phone`：联系电话
- `courseId`：关联课程 ID
- `status`：预约状态，取值为 `未确认`、`已确认`、`完成`
- `createTime`：预约创建时间
